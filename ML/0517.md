## lambda
```py
add=lambda x, y:x+y
sub=lambda x, y:x-y
mult=lambda x, y:x*y
div=lambda x, y:x/y if y!=0 else None

print(add(5,3))
print(sub(5,3))
print(mult(5,3))
print(div(5,3))
```
## map and filter
```py
number=[1,4,9,16,25]

squared=list(map(lambda x:x**2, number))
print(squared)
## [1, 16, 81, 256, 625]

even_number=list(filter(lambda x: x%2==0, number))
print(even_number)
## [4, 16]
```
